# Weather App

Simple weather application for the TI Badge that displays current weather conditions.

## Features

- Displays current weather for Boston (default)
- Shows temperature, conditions, wind, and humidity
- Uses wttr.in free weather service
- Refresh with 'R' key
- No API key required

## Usage

1. Launch the Weather app from the badge menu
2. Wait for weather data to load
3. Press 'R' to refresh
4. Press ESC to exit

## Requirements

- Network connection (WiFi or Ethernet)
- curl command-line tool (usually pre-installed)

## Future Enhancements

- Location selection via on-screen keyboard
- Save preferred location in config
- Multi-day forecast
- Weather icons

## Credits

Weather data provided by [wttr.in](https://wttr.in) - a console-oriented weather forecast service.
